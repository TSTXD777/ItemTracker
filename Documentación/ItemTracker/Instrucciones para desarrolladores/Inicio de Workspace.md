Requisitos para el desarrollo proyecto:

# 1. Instalación de Git:
Instalar Git, el cual es un sistema de control de versiones en desarrollo.
Página de Descarga: https://git-scm.com/install/windows

# 2. Creación de cuenta de GitHub
Para poder acceder al repositorio del proyecto es necesario tener una cuenta de GitHub, esto se realiza en la siguiente página web: https://github.com/

# 3. Instalación de GitHub Desktop
Para poder administrar los repositorios de manera local y mantener de manera sincronizada la información y los datos, se recomienda utilizar la Aplicación de Escritorio de GitHub, la descarga se encuentra en la siguiente página web: https://desktop.github.com/download/

# 4. Instalación de Obsidian
Para poder manejar y modificar la documentación de manera local, se hace uso de Obsidian, el cual es una herramienta para poder manipular archivos de texto, notas, vínculos, etc. La descarga se encuentra en la siguiente página: https://obsidian.md/download

# 5. Instalación de Visual Studio Code
Visual Studio Code en un IDE o editor de código donde vamos a programar la aplicación. El link de la página de descarga es: https://code.visualstudio.com/download

# 6. Instalación de Anaconda
Debido a que vamos a trabajar en python, tener un administrador de entornos nos ayuda a mantener nuestros paquetes y librerías en un solo lugar. La página de descarga es la siguiente: https://www.anaconda.com/download

# 7. Instalación de Godot
A pesar de que Godot en un Framework para diseñar videojuegos, en nuestro caso haremos uso de ello para poder crear el Frontend de la aplicación. Debido a que tiene una extensa habilidad de personalización, podemos hacer uso de ello para hacer una GUI o Interfaz de Usuario a libertad propia. Al descargar, asegurarse de descargar la versión .NET, la cual permite programación en C#. Link de descarga: https://godotengine.org/download/windows/
